# M3-RKD
“Multi-scale GNN-to-MLP Relational Knowledge  Distillation for Graph Classification”Code presentation
# Datsets
The datasets generated and analyzed during the current study are available in the [Zenodo repository] at https://doi.org/10.5281/zenodo.20151800
# Run
We provide the execution code for the MOLHIV and MOLBACE datasets. Once the environment is configured, you can simply run the experiments using bash MOLBACE-kd.sh and bash MOLHIV-kd.sh. The remaining experimental code will be made publicly available following the official publication of the paper.
